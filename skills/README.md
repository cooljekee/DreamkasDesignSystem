# Skills

Skill-файлы для Claude Code / Claude SDK, которые помогают потребителям Dreamkas Design System.

## dreamkas-ds-frontend

**Назначение.** Правила, антипаттерны и рабочие сниппеты для frontend-разработчика, который подключает Dreamkas DS к веб-проекту (React / Vue / чистый HTML). Содержит 5 критических правил, обнаруженных при реальной интеграции, плюс канонические структуры Sidebar / Topbar / App Shell и пошаговый setup.

**Как установить себе.** Скопируй папку `dreamkas-ds-frontend/` в `~/.claude/skills/` (либо в `.claude/skills/` своего проекта):

```bash
cp -r skills/dreamkas-ds-frontend ~/.claude/skills/
```

После этого Claude Code будет автоматически предлагать скилл, когда твоя задача связана с интеграцией ДС, переверсткой страниц на DS-классы, дебагом визуальных регрессий или созданием своих компонентов на DS-токенах.

**Привязка к ревизии.** Скилл опирается на структуру и имена токенов **rev 2**. При обновлении ДС читай раздел «Что делать, если ДС обновился» в `dreamkas-ds-frontend/SKILL.md`.

**Откуда правила.** Каждое правило в скилле — из конкретного кейса интеграции ДС rev 2 в проект «Антиштраф» (React 19 + Vite + Tailwind v4). Детальный разбор каждого кейса — в `../rev 2/FEEDBACK-2026-06-05.md`.

## Добавление нового скилла

1. Создай папку `<skill-name>/` рядом с `dreamkas-ds-frontend/`.
2. Положи внутрь `SKILL.md` с frontmatter:
   ```
   ---
   name: skill-name
   description: When to use this skill — короткое предложение для авто-обнаружения.
   ---
   ```
3. Допиши секцию про скилл в этот README.
