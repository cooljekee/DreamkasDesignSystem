Table Toolbar — Tablet
Tablet (тач-размеры). Шапка над таблицей: поиск по названию/ШК, фильтры-чипы, кнопка «Фильтры», переключатель плотности, экспорт и первичная кнопка. Плюс bulk-режим (выбрано N) — встроенный и плавающей панелью. Композирует Input, Button, Badge, Select.
