Modal — Mobile
Organism — overlay + диалог. Композирует Form Field, Button, Table, Alert. Размеры sm / md / lg, состояния loading / error, типы confirm / info / success. На телефоне модалки крепятся снизу как bottom sheet (см. вариант). Здесь — обзор типов.

Документ с данными — приёмка, инвентаризация. На телефоне сетка не нужна: каждая позиция — строка-карточка (№ · название/кол-во·цена · сумма справа). Читается легко, числа выровнены вправо.
Раскрывается как bottom sheet, header/footer прилипают, кнопки fluid в столбик. .doc-list

Асинхронное сохранение по сети. Тело под полупрозрачной вуалью со спиннером, кнопка «Сохранение…» с крутилкой, блокирует повторный submit.

Серверная или клиентская ошибка. Alert-баннер сверху тела + проблемные поля в состоянии error с сообщением под ними.

Organism = overlay + Card-обвязка + Form Field + Button + (Table / Alert внутри). Danger-подтверждение — без крестика. Кнопки футера справа: вторичная слева, primary справа.
